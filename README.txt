Launch Student Website Template
================================

This folder contains a simple 3-page website for Launch students.

Files
-----
- index.html - home page
- aboutme.html - about me page
- mytown.html - my town page
- styles.css - design and colors for all pages

What students should edit
-------------------------
Students should replace text inside square brackets, such as:

- [Your Name]
- [Town or City Name]
- [replace this]

Students can also edit the colors and font sizes in styles.css.

How to test
-----------
Open index.html in a browser, then click the navigation links.
